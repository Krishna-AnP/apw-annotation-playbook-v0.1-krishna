* APW Annotation Playbook – CVAT Cloud (Day 1)

# Overview
This repository documents hands-on work performed as part of the "APW Annotation Playbook" initiative.  
Day 1 focuses on understanding and using CVAT Cloud for image annotation, including project creation, task setup, annotation execution, and export.

# Objective
- Learn CVAT Cloud interface and workflow
- Create a project and task for image annotation
- Perform basic image annotations
- Export annotation results
- Maintain structured evidence and documentation

# Tool Used
- CVAT Cloud (https://app.cvat.ai)

# Work Performed (Day 1)
- Created a CVAT Cloud account and logged in
- Created a new project for image annotation
- Defined labels for the project
- Created a task using a local image dataset
- Uploaded 10 images for annotation (dummy data)
- Completed annotations for all images
- Exported annotation results
- Captured screenshots as evidence
