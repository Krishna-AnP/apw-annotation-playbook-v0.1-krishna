# Checklist – Spec Readiness Gate (v0.1)

This checklist must be completed before releasing work to annotators.

# Inputs Ready
- [ ] Label Specification document is created
- [ ] Label definitions are clear and unambiguous
- [ ] Edge Case Library is created
- [ ] Common edge cases are documented
- [ ] Annotation type is finalized

STOP-THE-LINE:
- [ ] Labels are unclear or missing
- [ ] Edge cases are not documented

# Process OK
- [ ] Label spec reviewed at least once
- [ ] No open questions from annotators
- [ ] Training quiz prepared
- [ ] Annotator training completed
- [ ] Annotator passed training quiz

STOP-THE-LINE:
- [ ] Annotator has not completed training
- [ ] Quiz not passed

# Outputs OK
- [ ] Label spec is frozen
- [ ] Edge Case Library finalized
- [ ] Batch Plan created
- [ ] Ready to release task for annotation
- [ ] Documentation stored in GitHub

STOP-THE-LINE:
- [ ] Label changes pending
- [ ] Batch plan missing
