# Checklist – Batch Release Gate (v0.1)

This checklist must be completed before releasing any batch for annotation.

# Inputs Ready
- [ ] Label specification approved
- [ ] Edge Case Library finalized
- [ ] Batch Plan created
- [ ] Dataset verified
- [ ] Annotator assigned

STOP-THE-LINE:
- [ ] Labels not frozen
- [ ] Batch plan missing

# Process OK
- [ ] Annotator trained and certified
- [ ] Training quiz passed
- [ ] CVAT task/job created
- [ ] Correct batch assigned

STOP-THE-LINE:
- [ ] Annotator not trained
- [ ] Wrong data assigned

# Outputs OK
- [ ] Batch released in CVAT
- [ ] Annotator notified
- [ ] Daily production report ready
- [ ] No blocking issues

STOP-THE-LINE:
- [ ] Batch released without approval
- [ ] Annotator not informed
